# AI Document Assistant

AI Document Assistant — сервис для интеллектуального поиска информации в документах. Пользователь загружает документы через веб-интерфейс, после чего они автоматически обрабатываются в фоновом режиме. После завершения обработки пользователь может задавать вопросы на естественном языке и получать ответы, основанные на содержимом загруженных документов.

## Возможности

* Авторизация пользователей.
* Загрузка документов через веб-интерфейс.
* Обработка документов с использованием Celery.
* Хранение оригиналов документов в MinIO.
* Сохранение метаданных документов в базе данных.
* Разбиение документов на чанки.
* Индексация чанков для последующего поиска.
* Ответы на вопросы пользователя по загруженным документам.
* REST API на базе FastAPI.

## Как работает система

### 1. Авторизация

Перед использованием системы пользователь проходит авторизацию.

### 2. Загрузка документа

Пользователь загружает документ через интерфейс приложения.

### 3. Фоновая обработка

После загрузки FastAPI отправляет задачу в Celery.

Во время обработки:

* сохраняются метаданные документа в базе данных;
* оригинальный файл сохраняется в MinIO;
* документ разбивается на небольшие текстовые чанки;
* чанки сохраняются в базе данных и подготавливаются для поиска.

### 4. Вопросы по документам

После завершения обработки пользователь может задавать вопросы.

Система:

1. находит наиболее релевантные части документа;
2. использует найденный контекст;
3. формирует ответ на основе содержимого документов.

## Основной сценарий использования

1. Пользователь входит в систему.
2. Загружает один или несколько документов.
3. Документы обрабатываются в фоне.
4. После завершения обработки пользователь задает вопрос.
5. Система находит релевантные части документов.
6. Пользователь получает ответ, основанный на содержимом документов.
